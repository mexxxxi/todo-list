# Todo-list

Консольное приложение для управления списком задач, написанное на C.

## Структура проекта

```
todo-project/
├── main.c       точка входа, обработка команд
├── task.c       реализация логики задач
├── task.h       заголовочный файл со структурами и объявлениями
├── Makefile     сборка проекта
└── db.txt       база данных задач (создаётся автоматически)
```

## Сборка

```bash
make
```

После сборки появится исполняемый файл `todo-list`.

## Формат данных

Задачи хранятся в текстовом файле `db.txt`, который создаётся автоматически при первом запуске.

Формат файла:
```
<количество задач>
<номер> <статус> <название>
<номер> <статус> <название>
...
```

- **Номер** — уникальный идентификатор задачи
- **Статус** — `0` (не выполнена) или `1` (выполнена)
- **Название** — текст задачи

Пример содержимого `db.txt`:
```
3
0 0 Buy a new notebook for coding
1 0 Fix memory leaks in database
2 1 Create a repository on GitHub
```

## Команды

| Команда | Описание |
|---|---|
| `add <название>` | Добавить новую задачу |
| `done <номер>` | Отметить задачу выполненной |
| `undone <номер>` | Вернуть задачу в статус «не выполнена» |
| `delete <номер>` | Удалить задачу |
| `list` | Вывести список всех задач |
| `clear` | Очистить все задачи (потребуется подтверждение) |
| `cld` | Очистить экран |
| `help` | Показать справку по командам |
| `exit` | Завершить работу программы |

Команды `done`, `undone` и `delete` принимают несколько аргументов:
```
done 0 1 2
delete 3 5
```

## Пример использования

```
add купить продукты
add прочитать книгу
list
============================================================
 number |                   name                   | status
============================================================
 0      | Buy a new notebook for coding            | TODO
 1      | Fix memory leaks in database             | TODO
============================================================
done 1
list
============================================================
 number |                   name                   | status
============================================================
 0      | Buy a new notebook for coding            | TODO
 1      | Fix memory leaks in database             | DONE
============================================================
delete 0
exit
```
