# Todo-list

Консольное приложение для управления списком задач, написанное на языке C.

## Структура проекта

```text
todo-project/
├── main.c       # Основной цикл обработки команд и ввода 
├── task.c       # Реализация функций базы данных и вывода 
├── task.h       # Объявление структур данных и прототипов функций 
├── Makefile     # Скрипт автоматической сборки проекта
├── README.md    # Описание проекта 
└── db.txt       # Файл хранения данных
